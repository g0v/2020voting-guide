# Numbering Legislator

This script is for numbering legislator

## Source Format

List of name.

example:

[
  "蔡英文",
  "郭台銘",
  "韓國瑜",
  "柯文哲",
  "張善政",
  "賴清德",
  "王金平",
  "朱立倫",
  "黃宏成台灣阿成世界偉人財神總統",
  "林國龍",
  "張亞中",
  "周錫瑋"
]

## Result Format

CSV of id, name

example:

id,name  
1,蔡英文  
2,郭台銘  
3,韓國瑜   
4,柯文哲  
5,張善政  
6,賴清德  
7,王金平  
8,朱立倫  
9,黃宏成台灣阿成世界偉人財神總統  
10,林國龍  
11,張亞中  
12,周錫瑋
